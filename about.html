CO2 World Emissions Data Visualization: Project Description


Justin Denk, Eric Tan, and Lu Yang

Data

Our dataset comes from the World Bank. Please see the included project write-up for more information.
URL: http://data.worldbank.org/indicator/EN.ATM.CO2E.PCmetric
Units: METRIC TONS PER CAPITA

Instructions

To specify a specific year's data to visualize, simply use the drop down on the lefthand panel. To start and stop an animation of all data from 1960 to 2010, use the toggle button.
Clicking on a specific country will display a line graph of that country's historical CO2 data.
